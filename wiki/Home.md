# Welcome to the Qore Wiki

Qore combines infrastructure discovery, secure credential management, and protocol-native resource management in a single binary. It runs entirely in the terminal with a keyboard-first interface and includes a built-in MCP server that exposes infrastructure operations to AI models without exposing credentials.
